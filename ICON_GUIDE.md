# Icon Resources Guide

## Installed Icon Libraries

### 1. **React Icons** (Currently Installed) ✅
- **Website**: https://react-icons.github.io/react-icons/
- **Collections**: Font Awesome, Material Design, Bootstrap Icons, Heroicons, and 40+ more
- **Usage**: `import { FaWifi } from 'react-icons/fa'` (Font Awesome)
- **Best for**: Comprehensive icon coverage with multiple styles

### 2. **Lucide React** (Currently Installed) ✅
- **Website**: https://lucide.dev/
- **Best for**: Clean, modern, consistent line icons
- **Usage**: `import { Wifi } from 'lucide-react'`

## Popular Icon Libraries Available via React Icons

### Font Awesome (fa)
- Most comprehensive icon set
- Examples: `FaWifi`, `FaSwimmingPool`, `FaCar`, `FaTv`, `FaUtensils`

### Material Design Icons (md)
- Google's Material Design icons
- Examples: `MdPool`, `MdHotTub`, `MdKitchen`, `MdLocalParking`

### Bootstrap Icons (bs)
- Clean, modern icons
- Examples: `BsWifi`, `BsSnow`, `BsFire`, `BsTv`

### Heroicons (hi, hi2)
- Beautiful hand-crafted icons
- Examples: `HiWifi`, `HiHome`, `HiSparkles`

### Simple Icons (si)
- Brand logos (Netflix, etc.)
- Examples: `SiNetflix`, `SiSpotify`

## How to Find Icons

1. **React Icons Website**: https://react-icons.github.io/react-icons/
   - Search by keyword (e.g., "pool", "wifi", "kitchen")
   - Browse by category
   - See previews of all icons

2. **Lucide Icons**: https://lucide.dev/icons/
   - Search by keyword
   - Clean, consistent style

3. **Font Awesome**: https://fontawesome.com/icons
   - Most comprehensive
   - Search by keyword

## Icon Selection Tips

- **Consistency**: Stick to one icon family per component for visual consistency
- **Size**: Icons should be recognizable at small sizes (24-32px)
- **Style**: Match icon style to your design (outline vs filled)
- **Meaning**: Choose icons that clearly represent the feature

The `Amenities.tsx` component now uses React Icons with Font Awesome for more illustrative icons.
